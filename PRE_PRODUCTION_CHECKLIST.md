# Pre-Production Checklist

- [ ] Apply the latest SQL in Supabase.
- [ ] Configure all Cloudflare environment variables.
- [ ] Test Stripe checkout and webhook confirmation.
- [ ] Test PayPal checkout and capture confirmation.
- [ ] Validate Supabase RLS policies.
- [ ] Test admin access and order management.
- [ ] Test customer and admin emails.
- [ ] Test mobile checkout, cart, shop, product, login, and admin flows.
- [ ] Confirm legal pages are present and accurate.
- [ ] Make a small real purchase.
- [ ] Confirm logs do not contain sensitive personal data or payment data.
